# PangoDemo

Modifications of code from <https://immortalsofar.com/PangoDemo/linux.png>.
Kindly provided by Mike Birch.  See <https://stackoverflow.com/a/30415800/2877364>.

# Running

    make && ./main

Requires GTK3.  On Ubuntu, `sudo apt install libgtk-3-dev`.
